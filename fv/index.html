Future Value

Future value is the value of an asset at a time in the future. It's useful as a means of estimate what investment assets will be worth in 5, 10, 25 years down the road. There's a lot of assumptions one has to make but it's helpful in determining how much money to put aside for retirement. At least, it has been for me.

As part of the first code for Code Beyond Work, I'm going to create a future value javascript module.
